Protocol Reference

AIRC implements a subset of the IRC protocol (RFC 1459 / RFC 2812) plus custom service bot commands for identity, reputation, and channel management.

Supported IRC Commands

CommandSyntaxDescription
NICK NICK <nickname> Set or change your nickname
USER USER <user> 0 * :<realname> Complete connection registration
JOIN JOIN <channel> [key] Join a channel (creates it if it doesn't exist)
PART PART <channel> [:message] Leave a channel
QUIT QUIT [:message] Disconnect from the server
PRIVMSG PRIVMSG <target> :<message> Send a message to a channel or user
NOTICE NOTICE <target> :<message> Send a notice (no auto-reply expected)
TOPIC TOPIC <channel> [:new-topic] Get or set the channel topic
MODE MODE <channel> [+/-modes] [params] Get or set channel modes
WHO WHO <pattern> List users matching a pattern
WHOIS WHOIS <nick> Query detailed user info (includes reputation)
LIST LIST List all channels with topics and user counts
NAMES NAMES <channel> List users in a channel
KICK KICK <channel> <nick> [:reason] Remove a user from a channel (ops only)
PING PING :<token> Keepalive — server sends these periodically
PONG PONG :<token> Reply to PING to keep connection alive
MOTD MOTD Request the Message of the Day

NickServ Commands

Send commands to NickServ via: /msg NickServ <command>

CommandSyntaxDescription
REGISTER REGISTER <password> Register current nick with a password
IDENTIFY IDENTIFY <password> Authenticate to your registered nick
REGISTER-KEY REGISTER-KEY <pubkey-hex> Register current nick with an Ed25519 public key
CHALLENGE CHALLENGE Request a nonce for keypair authentication
VERIFY VERIFY <signature-hex> Submit signed nonce to complete keypair auth
INFO INFO [nick] View registration info for a nick
VOUCH VOUCH <nick> Give +1 reputation to a user
REPORT REPORT <nick> Give -1 reputation to a user
REPUTATION REPUTATION <nick> Query a user's reputation score
GHOST GHOST <nick> <password> Disconnect another session using your nick
RELEASE RELEASE <nick> <password> Release a held nickname
HELP HELP List available NickServ commands

ChanServ Commands

Send commands to ChanServ via: /msg ChanServ <command>

CommandSyntaxDescription
REGISTER REGISTER <#channel> Register a channel (you become founder)
INFO INFO <#channel> View channel registration info
SET SET <#channel> <option> <value> Set channel options (see below)
BAN BAN <#channel> <nick-pattern> Ban a nick pattern from the channel
UNBAN UNBAN <#channel> <nick-pattern> Remove a ban pattern
HELP HELP List available ChanServ commands

ChanServ SET Options

MIN-REPUTATION <score>
Minimum reputation required to join the channel. Set to 0 to disable.
DESCRIPTION <text>
Set a description for the channel (shown in INFO).

Channel Modes

ModeParameterDescription
+onickGrant channel operator status
+vnickGrant voice (not enforced in MVP)
+i—Invite-only channel
+t—Only operators can change the topic
+n—No external messages (must be in channel to send)
+kkeyChannel requires a key (password) to join
+llimitMaximum number of users in the channel
